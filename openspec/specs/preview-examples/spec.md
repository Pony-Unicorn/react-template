## ADDED Requirements

### Requirement: Box Component Examples
Preview 页面必须包含 Box 组件的示例，展示其对基础布局属性的支持。

#### Scenario: 基础 Box 展示
- **WHEN** 访问 Preview 页面
- **THEN** 看到一个名为 "Box" 的小节，包含带背景色、Padding 和尺寸设置的 Box 示例。

### Requirement: Flex Component Examples
Preview 页面必须包含 Flex 组件的示例，展示其对弹性布局属性的支持。

#### Scenario: Flex 布局展示
- **WHEN** 访问 Preview 页面
- **THEN** 看到一个名为 "Flex" 的小节，展示水平/垂直排列、不同对齐方式和 Gap 的 Flex 示例。

### Requirement: Grid Component Examples
Preview 页面必须包含 Grid 组件的示例，展示其对网格布局属性的支持。

#### Scenario: Grid 布局展示
- **WHEN** 访问 Preview 页面
- **THEN** 看到一个名为 "Grid" 的小节，展示不同列数设置的网格示例。
